# YouTube Scraper

**Actor ID:** `streamers/youtube-scraper`

Scrape YouTube channels, videos, and search results for creator partnerships and lead discovery.

**Pricing:** $5.00 per 1,000 videos ($0.005 per result)

## Key Input Parameters

```json
{
  "searchQueries": ["Crawlee"],
  "maxResults": 10,
  "maxResultsShorts": 0,
  "maxResultStreams": 0,
  "downloadSubtitles": true,
  "subtitlesLanguage": "en"
}
```

| Parameter | Type | Description |
|-----------|------|-------------|
| `searchQueries` | array | Search terms for YouTube search (enter as you would in YouTube's search bar) |
| `startUrls` | array | Direct URLs to videos, channels, playlists, or hashtags. Supports video URLs, channel URLs, playlist URLs, and search results pages. |
| `maxResults` | integer | Max regular videos per search term or channel (default: 0, set to 0 to skip). Acts as limit for regular videos when scraping a channel. Max: 20,000 videos per URL. |
| `maxResultsShorts` | integer | Max YouTube Shorts per search term (default: 0). Only works with search terms, not URLs. |
| `maxResultStreams` | integer | Max live streams per search term (default: 0). Only works with search terms, not URLs. |

## Subtitle Download Options

| Parameter | Type | Description |
|-----------|------|-------------|
| `downloadSubtitles` | boolean | Download video subtitles and convert to selected format (default: false) |
| `saveSubsToKVS` | boolean | Save downloaded subtitles to key-value store instead of dataset (default: false). Requires downloadSubtitles to be enabled. |
| `subtitlesLanguage` | string | Language to download subtitles in: "any", "en", "de", "es", "fr", "it", "ja", "ko", "nl", "pt", "ru" (default: "en"). Requires downloadSubtitles to be enabled. |
| `preferAutoGeneratedSubtitles` | boolean | Prefer auto-generated subtitles over user-created ones (default: false). Requires subtitlesLanguage to be set. |
| `subtitlesFormat` | string | Subtitle format: "srt", "vtt", "xml", "plaintext" (default: "srt") |

## Search Filters

| Parameter | Type | Description |
|-----------|------|-------------|
| `sortingOrder` | string | Sort search results by: "relevance" (default), "rating", "date", "views" |
| `dateFilter` | string | Filter videos by upload date: "hour", "today", "week", "month", "year" |
| `videoType` | string | Filter by video type: "video" (default), "movie" |
| `lengthFilter` | string | Filter by video length: "under4" (< 4 min), "between420" (4-20 min), "plus20" (> 20 min) |

## Video Quality & Format Filters

| Parameter | Type | Description |
|-----------|------|-------------|
| `isHD` | boolean | Apply HD quality filter for search (default: false) |
| `is4K` | boolean | Apply 4K quality filter for search (default: false) |
| `is3D` | boolean | Apply 3D video filter for search (default: false) |
| `is360` | boolean | Apply 360-degree video filter for search (default: false) |
| `isHDR` | boolean | Apply HDR filter for search (default: false) |
| `isVR180` | boolean | Apply VR180 filter for search (default: false) |

## Content Filters

| Parameter | Type | Description |
|-----------|------|-------------|
| `hasSubtitles` | boolean | Apply Subtitles/CC filter for search (default: false) |
| `hasCC` | boolean | Apply Creative Commons filter for search (default: false) |
| `isLive` | boolean | Apply Live video filter for search (default: false) |
| `isBought` | boolean | Apply Purchased filter for search (default: false) |
| `hasLocation` | boolean | Apply Location filter for search (default: false) |

## Date Range (for Channels)

| Parameter | Type | Description |
|-----------|------|-------------|
| `oldestPostDate` | string | Scrape videos published after date. Absolute (e.g., "2024-05-03") or relative (e.g., "1 day", "2 days" for recent posts). Automatically resets sorting to NEWEST. Only works when scraping by URL. |
| `sortVideosBy` | string | Sort channel videos by: "NEWEST", "POPULAR", "OLDEST" (default: varies). Maps to sorting buttons on channel's Videos/Shorts/Live pages. Only works when scraping by URL. |

## Output Fields

### Basic Video Information

| Field | Description |
|-------|-------------|
| `id` | YouTube video ID |
| `title` | Video title |
| `url` | Full video URL |
| `thumbnailUrl` | Video thumbnail URL |
| `duration` | Video duration (formatted as HH:MM:SS or MM:SS) |
| `date` | Publication date (relative like "10 months ago" or absolute "2021-12-21") |
| `viewCount` | Video view count |
| `likes` | Number of likes |
| `commentsCount` | Number of comments |
| `text` | Video description text |
| `descriptionLinks` | Array of link objects from video description with url and text |

### Channel Information

| Field | Description |
|-------|-------------|
| `channelName` | Channel name |
| `channelUrl` | Channel URL |
| `channelDescription` | Channel description/bio |
| `channelDescriptionLinks` | Array of link objects from channel description with url and text |
| `numberOfSubscribers` | Channel subscriber count |
| `channelTotalViews` | Total views across all channel videos |
| `channelTotalVideos` | Total number of videos on channel |
| `channelJoinedDate` | Date channel was created (e.g., "Mar 18, 2015") |
| `channelLocation` | Channel location/country (if specified) |

### Video Features & Status

| Field | Description |
|-------|-------------|
| `isMonetized` | Boolean flag indicating if video/channel is monetized |
| `commentsTurnedOff` | Boolean flag indicating if comments are disabled |
| `location` | Video location/filming location (if specified) |

### Subtitles

Available when `downloadSubtitles` is enabled:

| Field | Description |
|-------|-------------|
| `subtitles` | Array of subtitle objects with: `srtUrl` (link to subtitle file in key-value store), `type` ("auto_generated" or "user"), `language` (language code), `srt` (full subtitle text in selected format) |

### Context Information

| Field | Description |
|-------|-------------|
| `fromYTUrl` | Original input URL that generated this result (useful when scraping multiple URLs or search terms) |
| `inputChannelUrl` | Original channel URL input (when scraping channel about page) |
